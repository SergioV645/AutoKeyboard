# Auto Spacebar 🖱️

A simple lightweight tool that automatically presses the **spacebar** every X seconds, with a clean GUI to control it.

## ⬇️ Download
👉 [Click here to download the latest version](https://github.com/SergioV645/auto-spacebar/releases/latest)

## ✨ Features
- Set a custom interval in seconds
- Simple Start / Stop buttons
- Status label showing current state
- Lightweight and easy to use

## 🖥️ How to use
1. Download the `.exe` from the link above
2. Open `AutoKeyboord.exe`
3. Set your desired interval (in seconds)
4. Click **▶ Start** and switch to the window you need
5. Click **■ Stop** whenever you want to stop

## ⚠️ Notes
- No installation required, just run the `.exe`
- Windows only
- Some antivirus may flag it as suspicious — this is a false positive caused by PyInstaller
